# OBS-Full-Window-Auto-Switcher
A LUA script that automates scene switching when opening and closing full-screen windows. Very useful for game streamers

**Supports Linux with X11 only!**

**Requires *xdotool* and *xprop*.**
To install, use the following commands:
```
sudo apt install xdotool x11-utils   # Debian/Ubuntu
# or
sudo pacman -S xdotool xorg-xprop    # Arch
```
